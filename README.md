# greenwood-lume

A demonstration repo of Greenwood and [**Lume**](https://lume.io/).

## Setup

1. Have Node >= 22.x (LTS) installed (or run `nvm use`)
1. Clone the repo
1. Run `npm ci`

Commands:

- `npm run dev` - start the dev server
- `npm start` - generate a production build and serve it
